# Video Production Guide
## Technische Spezifikationen

### Video-Format
- **Auflösung:** 1920x1080 (Full HD)
- **Frame Rate:** 30 fps
- **Format:** MP4 (H.264 codec)
- **Aspect Ratio:** 16:9
- **Bitrate:** 5-8 Mbps für gute Qualität

### Audio
- **Format:** AAC
- **Sample Rate:** 48 kHz
- **Bitrate:** 192 kbps
- **Kanäle:** Stereo

### Screen Recording Settings
- **Cursor:** Highlight aktivieren (gelber Ring)
- **Klicks:** Visuelles Feedback zeigen
- **Tastatureingaben:** Optional einblenden
- **System-Audio:** Aus (nur Voiceover)

## Empfohlene Tools

### Screen Recording
1. **OBS Studio** (kostenlos, professionell)
   - Scene-Switching möglich
   - Gute Qualität
   - Plugins für Cursor-Highlight

2. **Loom** (einfach, cloud-basiert)
   - Sehr benutzerfreundlich
   - Automatischer Upload
   - Integrierte Bearbeitung

3. **ScreenFlow** (Mac, €129)
   - Professionelle Bearbeitung
   - Animations-Effekte
   - Callouts und Zoom

### Video-Bearbeitung
1. **DaVinci Resolve** (kostenlos)
   - Professionelle Features
   - Color Grading
   - Text-Overlays

2. **Camtasia** (€249)
   - Speziell für Screen Recording
   - Einfache Bearbeitung
   - Viele Templates

### Voiceover
1. **Audacity** (kostenlos)
   - Audio-Recording
   - Noise Reduction
   - Normalisierung

2. **Adobe Audition** (Abo)
   - Professionelle Audio-Bearbeitung

### Optional: AI-Voiceover
- **ElevenLabs:** Sehr natürliche deutsche Stimmen
- **Google Text-to-Speech:** Kostenlos, gute Qualität
- **Amazon Polly:** Professionell, Pay-per-Use

## Produktions-Workflow

### Vorbereitung
1. **Testdaten vorbereiten**
   - Realistische Praxis-Daten erstellen
   - Sensible Informationen anonymisieren
   - Genug Beispieldaten für alle Features

2. **Browser vorbereiten**
   - Zoom auf 100%
   - Alle Benachrichtigungen deaktivieren
   - Cookies/Cache löschen für saubere Ansicht
   - Testbenutzer anmelden

3. **Workspace aufräumen**
   - Desktop aufräumen
   - Nur benötigte Apps offen
   - Second Monitor für Script

### Recording
1. **Test-Recording** (5-10 min)
   - Audio-Levels prüfen
   - Cursor-Highlight testen
   - Bildqualität checken

2. **Haupt-Recording**
   - Script bereithalten
   - Langsame, bewusste Bewegungen
   - Pausen zwischen Aktionen (für Schnitt)
   - Mehrere Takes wenn nötig

3. **B-Roll aufnehmen**
   - Alternative Ansichten
   - Detail-Aufnahmen
   - Zusätzliche Features

### Post-Production
1. **Schnitt**
   - Entferne Wartezeiten
   - Füge Übergänge ein (max. 0.5s)
   - Speed-Up für längere Ladezeiten

2. **Overlays hinzufügen**
   - Text-Overlays für Key-Points
   - Callout-Boxen für wichtige Elemente
   - Zoom für Details

3. **Audio**
   - Voiceover synchronisieren
   - Hintergrundmusik (leise, -20dB)
   - Sound-Effects für Klicks (optional)

4. **Captions**
   - Deutsche Untertitel für alle Voiceover
   - Große, lesbare Schrift
   - Kontrastreicher Hintergrund

5. **Branding**
   - Logo-Intro (3s max)
   - Logo-Watermark (dezent)
   - Abspann mit CTA

## Best Practices

### Dos
✓ Langsame, bewusste Cursor-Bewegungen
✓ Pausen lassen für Voice-Over
✓ Mehrere Takes aufnehmen
✓ Realistische Beispieldaten verwenden
✓ Key Features hervorheben
✓ Call-to-Action am Ende

### Don'ts
✗ Zu schnelle Bewegungen
✗ Echte Patientendaten zeigen
✗ Zu viele Features in einem Video
✗ Unvorbereitete Ad-hoc-Aufnahmen
✗ Lange Ladezeiten zeigen
✗ Fehler im UI zeigen

## Content-Strategie

### Video-Reihenfolge auf Landing Page
1. **KI-Praxisanalyse** - Hauptfeature, größtes Alleinstellungsmerkmal
2. **Workflow Automation** - Zeigt Zeitersparnis
3. **Team Management** - Wichtig für Praxisinhaber
4. **Recruiting System** - Spezialisiertes Feature

### Video-Hosting
**Empfehlung:** YouTube (unlisted) oder Vimeo
- Gute Performance
- CDN weltweit
- Embeddable
- Analytics

### Dateigrößen
- Ziel: < 20 MB pro Video (60s)
- Komprimierung nach Export
- Adaptive Streaming wenn möglich

## Beispiel-Zeitplan

### Produktion (pro Video)
- Vorbereitung: 1-2 Stunden
- Recording: 1-2 Stunden
- Editing: 2-3 Stunden
- Review & Feedback: 1 Stunde
- **Gesamt: 5-8 Stunden pro Video**

### Alle 4 Videos
- **Gesamt-Aufwand: 20-32 Stunden**
- Empfehlung: 1-2 Wochen einplanen

## Nächste Schritte

1. ☐ Screen Recording Tool auswählen und installieren
2. ☐ Testdaten für Demo-Praxis vorbereiten
3. ☐ Voiceover aufnehmen oder AI-Service wählen
4. ☐ Erstes Test-Video produzieren (z.B. KI-Praxisanalyse)
5. ☐ Feedback einholen und Prozess optimieren
6. ☐ Restliche 3 Videos produzieren
7. ☐ Videos auf Landing Page einbinden
8. ☐ Analytics einrichten für View-Tracking
