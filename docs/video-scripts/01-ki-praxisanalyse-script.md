# Demo Video Script: KI-Praxisanalyse (AI Practice Analysis)
**Duration:** 45-60 Sekunden
**Zielgruppe:** Ärzte, Praxisinhaber, MVZ-Leiter

## Szene 1: Problem Statement (0-10s)
**Visuell:** Dashboard mit vielen Daten
**Text Overlay:** "Verlieren Sie den Überblick über Ihre Praxisdaten?"
**Voiceover:** "Die meisten Praxen sammeln Daten, aber nutzen sie nicht optimal für Entscheidungen."

## Szene 2: AI Analytics Dashboard (10-25s)
**Visuell:** Navigation zu Analytics → AI Analysis Seite
**Highlights:**
- KPI-Übersicht mit farbcodierten Kennzahlen
- Interaktive Diagramme (Patientenzahlen, Umsatz, Auslastung)
- AI-generierte Insights anzeigen

**Voiceover:** "Effizienz Praxis analysiert Ihre Daten automatisch mit künstlicher Intelligenz. Sie erhalten konkrete Optimierungsvorschläge auf Knopfdruck."

**Screen Actions:**
1. Klick auf "AI-Analyse starten" Button
2. Zeige Live-Analyse-Animation
3. Zeige generierte Insights mit konkreten Empfehlungen

## Szene 3: Konkrete Insights (25-45s)
**Visuell:** Detail-Ansicht der AI-Insights
**Highlights:**
- "24% Effizienzsteigerung möglich" Badge hervorheben
- Konkrete Vorschläge zeigen (z.B. "Optimieren Sie Ihre Terminplanung")
- Prognose-Diagramme für nächste 3 Monate

**Text Overlays:**
- "Automatische Datenanalyse"
- "Konkrete Handlungsempfehlungen"
- "Bis zu 24% Effizienzsteigerung"

**Voiceover:** "Das System erkennt Muster in Ihren Daten und zeigt genau, wo Verbesserungspotenzial besteht."

## Szene 4: Call-to-Action (45-60s)
**Visuell:** Zurück zum Dashboard mit allen Features
**Text Overlay:** "Starten Sie jetzt Ihre kostenlose Demo"
**Voiceover:** "Erleben Sie KI-gestützte Praxisoptimierung. Jetzt Demo starten."

## Technische Notizen:
- **Cursor-Hervorhebung:** Ja, alle Klicks deutlich zeigen
- **Geschwindigkeit:** Langsame, klare Bewegungen
- **Übergänge:** Sanfte Fade-Transitions zwischen Szenen
- **Musik:** Ruhige, professionelle Business-Hintergrundmusik
- **Captions:** Deutsche Untertitel für alle Voiceover-Texte

## Zu zeigende Workflows:
1. Dashboard öffnen → Analytics navigieren
2. AI-Analyse starten mit echten Beispieldaten
3. Insights durchscrollen und Detail-View öffnen
4. Export-Funktion für Berichte zeigen
